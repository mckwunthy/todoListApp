################## APP DE GESTION DE TACHES########################

Description : application de gestion de tâches basique (ToDo List).

--> Ajout de Tâches :
	Ajouter de nouvelles tâches avec un titre et une description.

-->Affichage des Tâches :
	Afficher la liste des tâches enregistrées 

-->Gestion des tâches :
	- tâche en cours : marquée la tache en selectionnant "en cours" puis valider
	- tâche terminée : marquée la tache en selectionnant "terminée" puis valider

-->Suppression de Tâches :
	Supprimer des tâches de la liste.

--> Stockage des Données :
	Les tâches sont stockées dans un fichier.

Bon usage !
